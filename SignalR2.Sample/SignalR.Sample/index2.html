Quick steps to write a SignalR web application

    1. Create an "ASP.NET Empty Web Application" project
    2. Add NuGet packages:
        - Microsoft ASP.NET SignalR SystemWeb
        - Microsoft ASP.NET SignalR JS
    3. Configure SignalR
        - RouteConfig.cs
        - SignalRConfig.cs
        - Global.asax
    4. Create your data types (DataModel.cs)
    5. Create your SignalR Service (BackgroundThread.cs, SampleHub.cs)
    6. Create your SignalR Client (SampleHub.js, index.html)
    7. Launch web application and play with your real time app!
